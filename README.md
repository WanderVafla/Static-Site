# Project: Static Site Integration

This project is a pixel-perfect implementation of a desktop mockup (1920px) created for the **Jobtrek** training center.

## Technologies

* **HTML5**: Semantic and W3C valid structure.
* **CSS3**: Modern layout techniques using **Flexbox** and **Grid** (no floats).
* **Git**: Version control using **Conventional Commits** and GitHub Projects.

## Project Requirements

* **Pixel Perfect**: Site matches the design exactly at 1920x1080 resolution.
* **No JavaScript**: 100% pure HTML and CSS.
